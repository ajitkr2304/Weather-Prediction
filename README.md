# 🌦 Weather Prediction System

A modern and responsive Weather Prediction Web Application that provides real-time weather information using the WeatherAPI.

## 🚀 Live Demo

🔗https://weather-predicition-ajit-kumar04.netlify.app/

## 📂 GitHub Repository

🔗 https://github.com/Ajitkr2304/Weather-Prediction

---

## 📌 Project Overview

The Weather Prediction System is a responsive web application that allows users to search for any city and view current weather conditions along with a 3-day weather forecast.

The application automatically detects the user's current location (with permission) and displays the weather instantly.

---

## ✨ Features

- 🌍 Search weather by city name
- 📍 Auto Detect Current Location
- 🌡 Current Temperature
- ☁ Weather Condition
- 💧 Humidity
- 🌬 Wind Speed
- 🤗 Feels Like Temperature
- 📅 3-Day Weather Forecast
- 🖼 Dynamic Weather Icons
- 🎨 Dynamic Background
- 📱 Responsive Design
- ⚡ Fast Loading
- ❌ Error Handling


Open the project folder

```bash
cd Weather-Prediction
```

Open `index.html` in your browser.
